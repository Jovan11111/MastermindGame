Objective:
The objective of Mastermind is to guess the secret code created by the other player (or in the case of your app, the computer) within a limited number of attempts.
The Code:
The secret code consists of a sequence of colored pegs. The number of pegs and the range of colors used can vary depending on the specific version of the game.
Making a Guess:
Players take turns making guesses at what the code might be. After each guess, the other player provides feedback to help the codebreaker deduce the correct code.
Feedback:
The feedback consists of a series of key pegs that indicate the accuracy of the guess. A black peg indicates a correct color in the correct position, while a white peg indicates a correct color in the wrong position.
Winning the Game:
The game continues with the codebreaker making successive guesses until the code is correctly guessed or until the maximum number of allowed guesses is reached. The codebreaker wins by guessing the correct code within the given number of attempts.
